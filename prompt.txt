Un sistema web autoadministrable con tres módulos principales: 1) Inventario (con control de stock y alertas), 2) Recetas (con cálculo automático de costos) y 3) Ventas (con descuento de stock automático). Además, debe tener un Módulo de Contenido Digital para gestionar y etiquetar las fotos de productos. La interfaz debe ser limpia, responsiva y diseñada para el uso eficiente en dispositivos móviles. Como recomendacion me gustaria que el sistema tenga base de datos Mysql, que el frontend se haga con React, el testing con vitest y de ser posible mantener sencillo el backend con Node y express (sin librerias extra como NestJs salvo que sea muy beneficioso)
